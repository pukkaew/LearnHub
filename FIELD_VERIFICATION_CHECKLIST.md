## ส่วนที่ 3: TAB "ภาพรวม" (Overview)

### ✅ ต้องแสดง:

1. **วิดีโอแนะนำหลักสูตร**
   - [ ] แสดง YouTube embed (ถ้ามี intro_video_url)
   - [ ] หรือแสดงวิดีโอ Vimeo (ถ้าเป็น Vimeo URL)
   - [ ] ถ้าไม่มีวิดีโอ ไม่แสดงส่วนนี้

2. **เกี่ยวกับคอร์สนี้**
   - [ ] แสดงคำอธิบายแบบเต็ม (HTML formatted)
   - [ ] ข้อความ HTML ต้องแสดงผลถูกต้อง (ตัวหนา, ตัวเอียง, ฯลฯ)

3. **วัตถุประสงค์การเรียนรู้**
   - [ ] แสดงรายการ 5 ข้อ (bullet points):
     1. เข้าใจหลักการและแนวคิดพื้นฐานของเทคโนโลยีสารสนเทศ
     2. สามารถประยุกต์ใช้ความรู้ในการแก้ปัญหาเชิงปฏิบัติได้
     3. พัฒนาทักษะการคิดวิเคราะห์และการทำงานเป็นทีม
     4. มีความรู้ด้านเครื่องมือและเทคนิคที่ทันสมัย
     5. สามารถสื่อสารและนำเสนอผลงานได้อย่างมืออาชีพ

4. **กลุ่มเป้าหมาย**
   - [ ] แสดง **ตำแหน่ง:** (ในภาษาไทย)
     - developer → "นักพัฒนา"
     - engineer → "วิศวกร"
     - analyst → "นักวิเคราะห์"
     - manager → "ผู้จัดการ"
   - [ ] แสดง **แผนก:** (ในภาษาไทย)
     - IT → "ไอที"
     - Development → "พัฒนา"
     - Technology → "เทคโนโลยี"
     - Digital → "ดิจิทัล"

5. **ความต้องการพื้นฐาน**
   - [ ] แสดง "มีพื้นฐานการใช้คอมพิวเตอร์ และความเข้าใจเบื้องต้นเกี่ยวกับการเขียนโปรแกรม แนะนำให้มีประสบการณ์อย่างน้อย 1 ปี"

## ส่วนที่ 4: TAB "หลักสูตร" (Curriculum)

### ✅ ต้องแสดง:

1. **บทเรียนทั้งหมด (5 บท)**
   - [ ] แสดงหัวข้อ "บทเรียนทั้งหมด (5 บท)"
   - [ ] บทที่ 1: ความรู้พื้นฐานและการเตรียมความพร้อม
     - [ ] ระยะเวลา: 120 นาที
     - [ ] คำอธิบาย: แนะนำหลักสูตร ทำความรู้จักกับเครื่องมือ และเตรียมสภาพแวดล้อมการเรียนรู้
   - [ ] บทที่ 2: ทฤษฎีและหลักการสำคัญ
     - [ ] ระยะเวลา: 180 นาที
     - [ ] คำอธิบาย: เรียนรู้ทฤษฎีพื้นฐาน หลักการทำงาน และแนวคิดสำคัญ
   - [ ] บทที่ 3: การฝึกปฏิบัติและเทคนิคการใช้งาน
     - [ ] ระยะเวลา: 240 นาที
   - [ ] บทที่ 4: โปรเจกต์จำลองและกรณีศึกษา
     - [ ] ระยะเวลา: 300 นาที
   - [ ] บทที่ 5: โปรเจกต์จริงและการนำเสนอผลงาน
     - [ ] ระยะเวลา: 360 นาที

2. **การแสดงผล**
   - [ ] แต่ละบทมีเลขที่ลำดับ (1, 2, 3, 4, 5)
   - [ ] แสดงชื่อบทเรียน
   - [ ] แสดงคำอธิบาย (ถ้ามี)
   - [ ] แสดงระยะเวลา (นาที) พร้อมไอคอนนาฬิกา
   - [ ] มีไอคอน play ถ้ามีวิดีโอ

## การทดสอบหลังแก้ไข

### ถ้าข้อมูลแสดงไม่ถูกต้อง:

1. Screenshot หน้าจอที่แสดงผลผิด
2. บอกว่าฟิลด์ไหนแสดงผลไม่ตรง
3. บอกว่าแสดงอะไรแทน

### ตัวอย่างข้อผิดพลาดที่เคยพบ:

❌ ภาษา แสดง "ไทย" → ต้องแสดง "ภาษาไทย"
❌ ประเภท แสดง "ออนไลน์" → ต้องแสดง "บังคับ"
❌ ระดับความยาก แสดง "Intermediate" → ต้องแสดง "ปานกลาง"
❌ มีฟิลด์ "ราคา" → ต้องเอาออก
❌ จำนวนผู้เรียนสูงสุด แสดงผิด → ต้องแสดง "150 คน"

## สรุป: ฟิลด์ที่ต้องแสดงครบ

จากฟอร์มสร้างหลักสูตร (4 ขั้นตอน) ต้องแสดงฟิลด์เหล่านี้:

**Step 1 - ข้อมูลพื้นฐาน:**
1. course_name ✅
2. course_code ✅
3. category ✅
4. difficulty_level (แปลเป็นไทย) ✅
5. course_type (แปลเป็นไทย) ✅
6. language (แปลเป็นไทย) ✅
7. instructor_name ✅

**Step 2 - รายละเอียด:**
8. description (HTML) ✅
9. learning_objectives (array) ✅
10. target_positions (แปลเป็นไทย) ✅
11. target_departments (แปลเป็นไทย) ✅
12. prerequisite_knowledge ✅
13. duration_hours + duration_minutes ✅
14. max_enrollments (แสดงเป็น max_students) ✅

**Step 3 - เนื้อหา:**
15. course_image ✅
16. intro_video_url ✅
17. lessons[] (title, duration, description) ✅

**Step 4 - การประเมิน:**
18. passing_score ✅
19. max_attempts ✅
20. certificate_validity ✅
21. created_at ✅
22. updated_at ✅

**ฟิลด์ที่ไม่ต้องแสดงในหน้า detail:**
- ราคา (ระบบไม่ได้ใช้)
- enrollment_start/end (แสดงในส่วน enrollment status)
- test details (แสดงในหน้าอื่น)
